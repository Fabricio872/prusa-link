# Prusa Link
Widget for caching and showing data from web
## Installation
 1. go to plugin directory ([wordpress-dir]/wp-content/plugins)
 1. clone project
    ```
    git clone 
    ```
 1. go to project dir
    ```
    cd prusa-link
    ```
 1. install phantomjs 
    ```
    cp phantomjs /usr/local/bin/phantomjs
    ```
 1. install casperjs
    ```
    npm install -g casperjs
    ```
 1. install plugin dependencies
    ```
    composer install
    ```
